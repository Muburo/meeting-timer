# Meeting Timer

チーム会議の時間管理用ブラウザツール。発表者ごとの持ち時間をセットし、超過をアラートする。

## 使い方

1. `index.html` をブラウザで開く
2. プリセット（3min / 5min / 10min）で時間をセット
3. Start で開始

## 機能

- **円グラフ表示** — 残り時間を赤いリングで視覚的に表示。時間経過で減っていく
- **アラーム音** — タイマー終了時にビープ音で通知
- **色変化** — 残り1分で黄色、超過で赤色（マイナス表示でカウント継続）
- **ボタン状態表示** — 動作中は Start/Pause ボタンの有効/無効が切り替わる

## 技術

HTML + CSS + JavaScript のみ。サーバー不要、外部ライブラリなし。
